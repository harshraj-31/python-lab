# DSA Course Assignments

Python programs written for class tests and practical assignments.

## Folder guide

| Folder | Contents |
|---|---|
| `class-tests-and-practice/` | Individual class-test and practice programs (strings, dicts, file handling, menu-driven programs) |
| `gui-db-mini-projects/` | Six small Tkinter + MySQL mini-projects (patient management, cricket stats, library system, etc.) |
| `practical-sets/` | Two-question practical sets (SET-1 to SET-6), each pairing a data-structure question with a file-handling question |

## `class-tests-and-practice/`

| File | Description |
|---|---|
| `class_test_student_result_system.py` | Menu-driven student result system (add student, view result, report) |
| `mca_python_exercise.py` | String, list, and dictionary operations exercise |
| `practice_test.py` | List operations, string filtering, file sorting, states-seats menu program (needs `numbers.txt`) |
| `class_test_set1_word_frequency.py` | Word frequency analysis in a sentence |
| `set2_palindrome_check.py` | Palindrome checker |
| `ct1_set1_word_counter.py` | Word counter in a sentence |
| `ct1_set2_exam_processing.py` | Exam processing system |

## `gui-db-mini-projects/`

Each is a standalone Tkinter GUI backed by MySQL (or, for one, direct Excel/file I/O):

- `patient_management_system.py`
- `cricket_player_stats_system.py`
- `team_match_stats_system.py`
- `library_book_issue_system.py`
- `excel_orders_analytics_system.py`
- `hospital_billing_analytics_system.py`

These need a running MySQL instance with matching tables/credentials — check the
connection block near the top of each file before running.

## `practical-sets/`

Twelve files (SET-1 through SET-6, each with a Q1 and Q2), covering topics like
string processing, number operations, bank/library/medical-store systems, email
processing, train ticket booking, and product inventory.
